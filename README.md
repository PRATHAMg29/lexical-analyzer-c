# Lexical Analyzer using C

## Overview

This project implements a Lexical Analyzer in C that scans an input file and identifies lexical tokens. It performs token-based analysis by processing source code/text input and classifying different elements such as keywords, identifiers, operators, constants, and special symbols.

## Features

* Reads input from a file
* Performs lexical analysis
* Identifies and classifies tokens
* Processes keywords and identifiers
* Handles operators and special symbols

## Technologies & Concepts Used

* C Programming
* File Handling
* String Manipulation
* Character Processing
* Functions
* Modular Programming

## Project Structure

* `main.c` – Main execution flow
* `lexical.c` – Lexical analysis logic
* `main.h` – Function declarations and definitions
* `test.txt` – Sample input file

## How to Run

### Compile

```bash
gcc *.c 
```

### Execute

```bash
./lexical_analyzer test.txt
```
